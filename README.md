
Traffic jams waste time, fuel, and patience. This project explores how adaptive traffic signals can make intersections smarter by learning from real-time traffic flow. Using the SUMO (Simulation of Urban MObility) platform, we simulate city traffic and design controllers that respond dynamically instead of relying on fixed schedules.

**What this project does**
- Models realistic intersections and traffic scenarios in SUMO
- Applies adaptive logic to change signal phases based on vehicle density
- Compares performance against traditional fixed-time signals
- Tracks metrics like waiting time, queue length, and throughput

**Why it matters**
Adaptive traffic control is a step toward smart cities — reducing congestion, cutting emissions, and improving commuter experience. This project provides a sandbox for experimenting with algorithms ranging from simple heuristics to reinforcement learning.

**Future directions**
- Integration with reinforcement learning agents
- Multi-intersection coordination
- Real-world data integration for validation
